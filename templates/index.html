Wikipedia Chronological Search

This page is a demonstration of Wikipedia Chronological Search.
Enter a text and the application will search the Wikipedia, come up with a page, find top 10 important events from the page using TF-IDF and then display them in reverse chronological order (Latest comes first).
If there are multiple Wikipedia pages, you'll be asked to select one of the options from a dropdown.
Source code is publicly available at https://github.com/piyush-kgp/wikipedia-chronological-search
Technologies used:

  • Python Backend - flask, nltk, scikit-Learn, wikipedia
  • HTML Frontend - JQuery, JS
